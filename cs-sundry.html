Sundry Markets

Sundry Markets is a fast-growing e-commerce platform specializing in curated grocery and specialty goods. As their business expanded, they faced challenges in managing their legacy infrastructure, scaling during peak demand, and ensuring a seamless customer experience. To modernize their digital presence, Sundry Markets partnered with Softgem to build their e-commerce website and mobile app using cloud-native technologies, DevOps practices, and a scalable architecture.

Industry Vertical


Ecommerce

The Challenges


Legacy Infrastructure Limitations:

  • Monolithic architecture led to slow deployments and difficulty in scaling.
  • High downtime during traffic spikes.

Slow Development Cycles:

  • Manual deployments caused delays in feature releases.
  • Lack of CI/CD pipelines increased human errors.

Security & Compliance Risks:

  • PCI-DSS compliance for payments was not fully optimized.
  • Vulnerability to DDoS attacks and data breaches.

To overcome these challenges, Sundry Markets needed:


❖ Scalability & Performance: Ensure the platform can handle high traffic, especially during sales and holiday seasons.
❖ Faster Deployment Cycles: Reduce time-to-market for new features and updates.
❖ Seamless Omnichannel Experience: Unify web and mobile app experiences for customers.
❖ Cost Optimization: Improve infrastructure efficiency while reducing operational costs.
❖ Enhanced Security & Compliance: Implement robust security measures for payment processing and user data.

Solution


Softgem proposed a cloud-native transformation using AWS for scalability, Key components delivered include:

Migration to Microservices:
- Decomposed the monolithic app into scalable microservices.
- Used AWS EKS for container management.
CI/CD Automation:
- Implemented GitHub Actions & AWS CodePipeline for automated testing and deployments.
Performance Optimization:
- Used Amazon CloudFront CDN for faster content delivery.
- Implemented Redis caching for high-speed data retrieval.
Security Enhancements:
- AWS WAF & Shield for DDoS protection.
- PCI-DSS compliant payment processing.

Results

  • Improved Scalability & Uptime: Auto-scaling reduced infrastructure costs by 30%.

  • Enhanced Customer Experience: Unified cart & checkout across web and mobile.

  • Cost Optimization: Reduced server costs by 25% with AWS Spot Instances and automated scaling saved infrastructure costs.

  • Security & Compliance: Achieved PCI-DSS Level 1 compliance. and Zero security breaches post-migration.

  • Monitoring & Alerting: AWS CloudWatch for real-time performance tracking, Sentry & New Relic for error monitoring. and Automated alerts for downtime & security threats.
