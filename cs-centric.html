Centric Platforms

Centric Platforms is a technology company that provides over 100 APIs, allowing businesses to integrate financial data and secure payment systems into their products. Since its launch in 2020, Centric has grown to support over 1,000 subscribers and works with more than 30 contributors. Their mission is to create efficient APIs that help clients develop products faster.

Industry Vertical


Fintech and API Integration Services

Solution


  • To address these challenges, SoftGEM Global Technologies LTD helped Centric migrate to AWS using the following services:

    Amazon EC2 and VPC:These services provided scalable computing power and secure network infrastructure, allowing Centric to handle increased traffic and ensure data safety.
  • Amazon API Gateway and AWS Lambda: These tools helped manage API traffic and handle demand spikes. AWS Lambda, a serverless computing service, allowed Centric to scale resources automatically based on the traffic without the need to manage servers.

  • Amazon RDS and DynamoDB: These database solutions helped Centric manage both structured data (RDS) and fast-moving unstructured data (DynamoDB), offering reliable and efficient data storage.

  • ● AWS CloudWatch:This service provided real-time monitoring of Centric's system, allowing them to track performance and identify issues quickly.

  • AWS WAF and IAM:AWS WAF (Web Application Firewall) protected Centric from malicious traffic, while IAM (Identity and Access Management) helped manage secure access to the system.

Results

  • Improved Scalability: Centric's platform saw a 200% increase in API calls within the first three months of migration, without experiencing any downtime or performance degradation. The scalability features of Amazon EC2 and Amazon VPC allowed Centric to handle up to three times its original user base while keeping the platform responsive.

  • Better Performance: With the use of Amazon API Gateway and AWS Lambda, the time taken for API responses reduced by 50%, improving overall user experience. After migration, the platform's uptime improved, reaching 99.99% availability** due to the high-performance architecture provided by AWS.

  • Cost Savings: By using AWS Lambda serverless and EC2 auto-scaling, Centric was able to scale resources up or down based on actual demand. This dynamic scaling reduced the need for constant provisioning of idle resources, cutting down operational costs by 40%.

  • The serverless architecture allowed Centric to eliminate overprovisioning costs. They only paid for the compute resources they used, reducing unnecessary infrastructure expenses.

  • Stronger Security: Post-migration, Centric experienced a 100% improvement in security with AWS WAF and IAM, preventing data breaches and unauthorized access attempts. Faster Compliance Audits: Centric was able to meet regulatory compliance standards faster, with AWS tools simplifying security audits and data protection protocols.
